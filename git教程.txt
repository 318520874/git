1.新建版本库
	a.创建目录 mkdir learngit
	b.cd learngit
	c.显示当前目录 pwd
	d.把目录变为仓库 git init
2.文件添加到版本库
	a.文件添加到仓库 git add 文件名
	b.提交文件版本 git commit -m "说明"
3.git status 掌握仓库当前状态，知道哪些文件被修改
4.git diff 文件名 查看文件具体被修改的内容
